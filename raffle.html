Weighted Random Selection

Selecting a random item given a list of items and their weights

Instructions

This simple web app will randomly pick a raffle winner, given a list of entrants along with the number of raffle tickets belonging to each entrant. Upload a Microsoft Excel (.xlsx) file that conforms to the following formatting:

  • The below data is all on the first sheet
  • There are no blank or title rows preceeding the data: the relevant data begins at Row 1
  • Column A contains the entrant's first name
  • Column B contains the entrant's last name
  • Column D contains the number of raffle tickets belonging to the entrant
  • Other sheets and columns are allowed to contain irrelevant data, but it will not be processed

Upon uploading a valid file and clicking the "Select Winner" button, the winner's full name will appear below.
